## **react 기초**
- jsx에서 컴포넌트로 내보낼때는 하나의 박스 형식으로 만들어 줘야함
```js
function App() {
  return (
    // 이렇게 div 혹은 빈 상자로 감싸줘야함
    <div className="App"> 
      <header></header>
      <section></section>
      <article></article>
      <footer></footer>
    </div>
  );
}
// 위의 App 컴포넌트를 부를 때
export default App;
// 이 큰 덩어리가 컴포넌트
```

- js의 기능을 쓰기 위해서는 {} 안에 담아서 호출한다
